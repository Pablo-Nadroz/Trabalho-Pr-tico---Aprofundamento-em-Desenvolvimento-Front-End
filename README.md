# Trabalho Prático - Aprofundamento em Desenvolvimento Front-End

Este repositório contém as práticas desenvolvidas para a disciplina de Aprofundamento em Desenvolvimento Front-End.

## Conteúdo

### 1. Práticas JavaScript (Exemplos JS)
Implementação de algoritmos de ordenação e um catálogo dinâmico de receitas utilizando HTML, CSS (Bootstrap) e JavaScript puro. Inclui também o consumo de uma API REST pública (Reqres) utilizando Vue.js.

### 2. Projeto Angular (Catálogo de Livros)
Aplicação Angular (`livros-angular`) desenvolvida de acordo com os procedimentos do curso, contendo:
- Componentes e Rotas (Lista de Livros e Cadastro de Livros).
- Serviços e Injeção de Dependências.
- Formulários com validação.
- Interface construída com o Bootstrap.

## Como Executar

**Para o projeto Angular:**
1. Navegue até a pasta `livros-angular`:
   ```bash
   cd livros-angular
   ```
2. Instale as dependências (caso não estejam instaladas):
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   npm run start
   ```
4. Acesse `http://localhost:4200/` em seu navegador.
